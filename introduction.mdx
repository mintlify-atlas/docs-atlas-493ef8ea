## What is VITACORE?

VITACORE is a web-based **Electronic Health Record (EHR)** system built with Flask and Supabase. It provides healthcare organizations with a centralized platform to manage patient records, track clinical evolutions, record vital signs, and administer healthcare professionals across multiple locations.

## Technology stack

| Component | Technology |
|-----------|-----------|
| Backend framework | Python / Flask |
| Database & Auth | Supabase (PostgreSQL + Auth) |
| Session management | Flask server-side sessions |
| Frontend | Jinja2 HTML templates |
| Environment | python-dotenv |

## How the system is organized

VITACORE uses Flask **Blueprints** to separate concerns:

- **`/auth`** — Login and logout
- **`/hc`** — Dashboard
- **`/hc/pacientes`** — Patient CRUD
- **`/hc/evoluciones`** — Clinical evolution notes
- **`/hc/signos`** — Vital signs
- **`/hc/configuracion`** — System configuration (sedes, consultorios, especialidades, profesionales, servicios, países, tipos de documento)
- **`/hc/configuracion/roles-permisos`** — Role and permission management
- **`/hc/configuracion/usuarios`** — User account management
